## Commit convention

| 태그이름 | 설명                                     |
| -------- | ---------------------------------------- |
| feat     | 새로운 기능 추가                         |
| fix      | 버그 수정                                |
| design   | 디자인 변경                              |
| style    | 코드 포맷팅, 세미콜론누락, 코드 변경없음 |
| refactor | 코드 리팩토링                            |
| test     | 테스트 코드, 리펙토링 테스트 코드 추가   |
| rename   | 파일, 폴더명 수정 및 위치 변경           |
| remove   | 파일 삭제                                |
| docs     | 문서 수정                                |
| merge    | 머지                                     |
